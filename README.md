# ZEBRA-shop

## Getting Started

To get started, you can follow these steps:

1. Install the project dependencies using npm:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm start
   ```

It will start listening on Port 5173.

## Technologies Used

1. For frontend:
   *React
   *Vite
   \*TypeScript

2. For server:
   *Node.js
   *Express.js
   \*MongoDB
